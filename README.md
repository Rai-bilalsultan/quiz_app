# Flutter Quiz App

A simple Quiz Application built with Flutter that displays True/False questions. Users can answer each question one by one, and after completing all questions, they are given the option to restart the quiz.

## Features

- 📱 Simple and clean user interface
- ✅ True/False quiz questions
- 🔄 Automatic navigation to the next question
- 🎉 Quiz completion dialog
- 🔁 Restart quiz functionality
- ⚡ Built using Flutter StatefulWidget

## Technologies Used

- Flutter
- Dart
- Material Design

## Project Structure

```
lib/
│── main.dart
```

## How It Works

1. The app starts by displaying the first question.
2. The user selects either **True** or **False**.
3. The app automatically moves to the next question.
4. After the last question, a dialog appears indicating that the quiz has been completed.
5. Pressing **Restart** resets the quiz and starts from the first question.

## How to Run

### Prerequisites

- Flutter SDK installed
- Android Studio or VS Code
- Android Emulator or Physical Device

### Steps

1. Clone the repository

```bash
git clone https://github.com/yourusername/flutter_quiz_app.git
```

2. Navigate to the project folder

```bash
cd flutter_quiz_app
```

3. Get the required packages

```bash
flutter pub get
```

4. Run the application

```bash
flutter run
```
